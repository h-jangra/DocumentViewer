# Document Viewer

A simple Chrome extension that opens documents in the browser instead of downloading them.

## Features
- Opens supported documents in a new tab
- Prevents forced downloads
- Uses the browserâ€™s built-in or simple custom viewers
- Works automatically on supported links

## Supports
PDF, DOCX, PPTX, XLSX, TXT

## Install
1. Open `chrome://extensions`
2. Enable Developer mode
3. Click **Load unpacked**
4. Select the project folder
